# Context Loader

Read this file first to determine which context to load.

## Always Load

- `context/IDENTITY.md`
- `context/MEMORY-SYSTEM.md`

## Task Context

For the active task:
- `context/tasks/task-{id}/task.md`
- `context/tasks/task-{id}/memory.md` (recent)

## Global Context

- `context/tasks/memory.md` (recent)
- `context/CHAT.md` (recent)

If required context is missing or ambiguous, ask the user.
