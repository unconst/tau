# Tau

A chat agent that learns from you, writes its own software to upgrade itself, and solve problems for you. 

## How to install?

```bash
curl -fsSL https://raw.githubusercontent.com/unconst/tau/main/install.sh | bash
```

## How to use?

1. Typing a `<prompt>` calls the agent normally.
2. Use `/adapt <prompt>` to update the agent code and restart it.
3. Use `/task <prompt>` to add a longer running challenge which the agent comes back to periodically.
